基于ThinkPHP 5.10版本。

背景：
>本人学习ThinkPHP 5，练手项目。
>部署在阿里云上。
>提高个人效率。
>方便书友朋友的交流。

基础架构：
>php 5.5.36
>Mysql 5.7.18
>Nginx 1.12.1
>ThinkPHP 5.10

核心功能：
+效率部分：
>tasklist
>password storage
>book list
>聚合搜索
>小工具
+读书交流部分：
>我读（分享读书笔记、评价）
>我问（以书中内容为出发点，提问）
>我思（拓展思维，形成个人观点）

欢迎交流学习，共同进步！
